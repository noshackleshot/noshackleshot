## Repository Overview

This is a GitHub profile repository (noshackleshot/noshackleshot) that displays on the user's GitHub profile page. The repository contains a single README.md file with a modern landing page design.

## Key Files

- **README.md** - The main profile page displayed on GitHub, styled as a modern black/white tech landing page in INFATIUM aesthetic

## Development

This repository has no build process, tests, or compilation steps. The README.md uses inline HTML and CSS for styling, which is rendered directly by GitHub.

### Editing the README

- Use inline HTML with CSS styles (no external stylesheets)
- GitHub supports HTML and CSS, but not JavaScript
- Use SVG for graphics and icons
- CSS hover effects work for interactivity
- Keep the black (#0a0a0a) / white (#ededed) color scheme
- Maintain the minimalist, strict aesthetic

## Design System

The README follows the INFATIUM design system:
- **Colors**: Black background (#0a0a0a), white text (#ededed)
- **Typography**: Large headlines with light body text
- **Spacing**: Generous padding and gaps
- **Effects**: Ultra-subtle backgrounds (2-5% opacity), glassmorphism, hover transitions
- **Language**: Russian
